# Capstone Artefacts

Phase-end deliverables that prove mastery. Organised as `artefacts/<topic>/phase-N/`.

The rule: a phase isn't closed until a concrete artefact exists here that a peer could look at and understand what was learned. Not notes, not summaries — something you could show.

Each topic's plan file names the expected capstone per phase in a "Capstones by Phase" table near the top.

On topic completion, leave this directory in place — it's the permanent record of what was built.
